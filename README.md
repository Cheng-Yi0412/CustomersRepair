# 客戶維修與銷售紀錄管理系統



## 技術棧 (Tech Stack)

- \*\*Frontend:\*\* Vue 3, TypeScript, Vite, Bootstrap 5

- \*\*Backend:\*\* .NET 8 Web API, Entity Framework Core

- \*\*Database:\*\* SQL Server


## 🛠️ 目前已完成功能

- 客戶資料列表顯示 (GET API 串接)

- 雙擊資料列開啟 Modal 編輯視窗

- RESTful API 規格實作 (PUT 更新功能)
